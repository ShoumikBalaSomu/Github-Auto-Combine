📖 Setup Guide

📱 How to use with OTT Navigator

  1. Copy the playlist link — Click "Copy Link" on any playlist above
  2. Open OTT Navigator — Launch the app on your device
  3. Add Provider — Go to Settings → Providers → Add Provider
  4. Paste the URL — Select "M3U Playlist" and paste the copied link
  5. Save & Refresh — The channels will load automatically, organized by country

➕ How to Add Your Playlists

  1. Edit playlists.txt — Go to the input/playlists.txt file in the repository
  2. Add URLs — Add your M3U/M3U8 playlist URLs, one per line
  3. Commit & Push — Save and push your changes
  4. Auto-merge — GitHub Actions will automatically download, merge, deduplicate, and organize your playlists
  5. Get your link — Use the same playlist URL above — it's always up to date!
